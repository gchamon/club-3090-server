Model Presets

Discovered presets are rendered directly from the local /opt/ai/club-3090 clone. Global applies single-GPU presets across every GPU, dual presets across every two-GPU pair, and multi-GPU presets to the shared runtime. Use the Custom Model menu item to bridge upstream pull.sh into this panel without changing the core UI flow.

API Presets

Default presets are locked. Custom presets are exposed as :8009/v1/<name> and :8009/<name>. Both forms work with short- and concise- prefixes.

Custom Preset Templates

Create custom endpoint templates for different workloads or clients. Each preset saves generation parameters like temperature, sampling, thinking mode, penalties, and token limits, then exposes them as custom OpenAI-compatible endpoints such as :8009/v1/my_preset and :8009/my_preset. Short and concise prefixes work with custom presets too.
